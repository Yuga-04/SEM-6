# Project Appraisal

Project appraisal is the process of evaluating a proposed project to determine its feasibility, profitability, and viability before investment. It helps entrepreneurs and financial institutions decide whether a project should be accepted or rejected. 

## Objectives of Project Appraisal

* To check profitability of the project
* To evaluate technical and financial feasibility
* To reduce investment risk
* To help in decision-making
* To ensure proper utilization of resources



# Methods of Project Appraisal

The methods are classified into:

1. Traditional Methods (Non-Discounting Methods)
2. Modern Methods (Discounted Cash Flow Methods)



# I. Traditional Methods

These methods do not consider the time value of money.

# II. Modern Methods

These methods consider the time value of money. 

## 3. Net Present Value (NPV)

NPV is the difference between present value of cash inflows and investment cost.

### Decision Rule

* Positive NPV → Accept project
* Negative NPV → Reject project

### Merits

* Considers time value of money
* Uses all cash flows

### Demerits

* Calculation is complex



## 4. Internal Rate of Return (IRR)

IRR is the discount rate where present value of inflows equals investment cost.

### Decision Rule

* IRR > Required return → Accept
* IRR < Required return → Reject

### Merits

* Scientifically reliable
* Considers full project life

### Demerits

* Difficult to calculate

# Conclusion

Project appraisal helps small-scale enterprises evaluate investment proposals and select profitable projects. Traditional methods are simple, while modern methods provide more accurate results by considering time value of money and future cash flows.
