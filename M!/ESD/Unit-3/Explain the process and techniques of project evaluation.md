# 1. Process of Project Evaluation

The project evaluation process involves the following steps:

### 1. Identification of Investment Opportunity

The organization first identifies a potential project or investment opportunity that can generate future returns.

### 2. Estimation of Cash Flows

The expected **cash inflows and outflows** of the project are estimated over its lifetime.
These include:

* Initial investment cost
* Operating costs
* Expected revenues
* Maintenance expenses

### 3. Determining Cost of Capital

The **cost of capital** represents the minimum required rate of return for the project.
It is used as the **discount rate** in evaluation techniques.

### 4. Evaluation Using Capital Budgeting Methods

Different financial techniques are applied to determine whether the project is profitable.

### 5. Selection of the Best Project

Projects are compared and the one with **higher profitability and lower risk** is selected.

### 6. Implementation and Monitoring

After selection, the project is implemented and its performance is monitored to ensure it meets expected results.

# 2. Techniques of Project Evaluation

The PDF classifies project evaluation techniques into **two main categories**. 

## 1. Traditional Methods (Non-Discounting Methods)

These methods are based on accounting information and **do not consider the time value of money**. 

### a) Payback Period Method

The **payback period** is the time required for an investment to recover its initial cost.

**Formula:**

Payback Period =
Initial Investment / Annual Cash Inflow

**Features:**

* Simple and easy to calculate
* Focuses on liquidity and quick recovery
* Lower payback period indicates a better investment

**Limitations:**

* Ignores time value of money
* Does not consider cash flows after payback period

### b) Accounting Rate of Return (ARR)

ARR measures the **average profit earned from an investment compared to the average investment**.

**Formula:**

ARR = Average Annual Income / Average Investment

**Decision Rule:**

* If ARR is higher than the required rate → Accept the project
* If ARR is lower → Reject the project

**Limitations:**

* Ignores time value of money
* Does not consider project life

## 2. Modern Methods (Discounted Cash Flow Methods)

These methods consider the **time value of money**, meaning money today is worth more than money received in the future. 

### a) Net Present Value (NPV)

NPV calculates the **difference between the present value of future cash inflows and the initial investment**.

**Formula:**

NPV = Present Value of Cash Inflows – Initial Investment

**Decision Rule:**

* NPV > 0 → Accept the project
* NPV < 0 → Reject the project

**Advantages:**

* Considers time value of money
* Uses all cash flows of the project
* Helps maximize shareholders’ wealth

### b) Internal Rate of Return (IRR)

IRR is the **discount rate at which the present value of cash inflows equals the initial investment**.

In other words, it is the rate where:

Present Value of Cash Inflows = Initial Investment

**Decision Rule:**

* If IRR > required rate of return → Accept
* If IRR < required rate → Reject

**Advantages:**

* Considers time value of money
* Helps compare multiple projects

### c) Profitability Index (PI)

Profitability Index measures the **ratio between the present value of future cash inflows and the initial investment**.

**Formula:**

PI = Present Value of Future Cash Inflows / Initial Investment

**Decision Rule:**

* PI > 1 → Accept the project
* PI < 1 → Reject the project

# Conclusion

Project evaluation is an essential process for selecting profitable investments. It involves estimating cash flows and applying financial techniques such as **Payback Period, ARR, NPV, IRR, and Profitability Index**. Among these methods, **discounted cash flow techniques like NPV and IRR are considered more reliable** because they account for the **time value of money and overall project profitability**.
